## :wrench: Instalación

1. **Clonar el repositorio**
	```bash
	git clone https://github.com/aMaicol/practice_02.git
	cd practice_02
	```

2. **Crear y activar un entorno virtual (.venv)**
	```bash
	python -m venv .venv
	. .venv/bin/activate
	```

3. **Instalar las dependencias**
	```bash
	pip install -r requirements.txt
	```
